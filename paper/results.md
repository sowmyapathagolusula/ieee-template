# Results

## Experimental Setup

The proposed AI-Powered Zero Trust Security Dashboard was developed and tested in a simulated software organization environment. The system collected user activity logs, repository access records, file download events, and login information to evaluate user behavior. AI-based anomaly detection techniques were applied to identify suspicious activities and assess system performance.

## Performance Metrics

The performance of the proposed system was evaluated using the following metrics:

- **Detection Accuracy:** Measures the system's ability to correctly identify anomalous and normal activities.
